编译命令

  • 运行 server

    react-native start 或者 npm start

  • 运行 客户端

    react-native run-android

  • 安装模块,添加依赖

    npm i react-native-swiper --save

  • 查看模块

    npm view react-native-swiper

  • 删除模块

    npm rm react-native-swiper
